# Tutorial: Create and Use a Custom Configuration File

+ Create a new `YAML` configuration file under folder *configuration*, like *custom.yml*.  
+ Copy everything in *default.yml* into *custom.yml* :   

> **custom.yml:**  
> 　　player-genetic-option:  
>  　　　　maxHealth: 40　　# Int: Player's Basic Max Health    
> 　　// ......  


　　**WARNING: Please DO NOT modify anything in default.yml**  


+ Modify the parameters in * custom.yml * as needs.  
+ When preset a battlefield, use command **/battleroyale preset [worldName] custom** to use your configuration file.
